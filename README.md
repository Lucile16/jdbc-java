# jdbc-java

### Pour le TP5 :
1) Pour exécuter les tests, rester sur la branche main.<br>
2) Ensuite, double-cliquez sur un fichier de tests au choix parmi :
   - Main
   - TestConnectionJdbc
   - TestDelete
   - TestInsertion
   - TestSelect
   - TestUpdate<br>

3) Enfin, vous pouvez lancer le test en cliquant sur le bouton "Run" en haut à droite.

3) Enfin, vous pouvez lancer le test en cliquant sur le bouton "Run" en haut à droite.
